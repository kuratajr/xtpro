# FAQ ❓

## xtpro là gì?

xtpro là bộ **client/server** để tunneling (HTTP/TCP/UDP) và file sharing (WebDAV).

## Chạy client thế nào?

```bash
./bin/client/xtpro-linux-amd64 --proto http 3000
./bin/client/xtpro-linux-amd64 22
```

## Chạy server thế nào?

```bash
./bin/server/xtpro-server-linux-amd64
```
