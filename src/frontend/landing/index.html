XTPro quickstart 
# 1. HTTP tunnel (server tự cấp subdomain HTTPS)
xtpro --proto http 3000

# 2. Public SSH (TCP mặc định)
xtpro 22

# 3. Mở nhiều tunnel trong 1 lệnh (multi-port)
# Cú pháp: [proto:]local[:remoteRequest]
xtpro tcp:2222:11111 udp:53 http:3000 3389

# 4. Kết nối tới server riêng của bạn
xtpro --server your-vps.example.com:8882 --ports "http:3000 tcp:22 udp:19132"

Chỉ cần một lệnh để đưa dịch vụ của bạn ra toàn cầu.

Đăng ký miễn phí, nâng cấp khi bạn cần nhiều tunnel hơn, thống kê chi tiết hơn hoặc quyền quản trị đội nhóm.
